💾 Converting to PDF

Method 1: Using the Print Button (Easiest)

  1. Click any documentation link above
  2. Click the blue "🖨️ Print / Save as PDF" button in the top-right corner
  3. In the print dialog, select "Save as PDF" as the destination
  4. Click "Save" and choose your location

Method 2: Using Browser Menu

  1. Open any HTML file
  2. Press Ctrl+P (Windows/Linux) or Cmd+P (Mac)
  3. Select "Save as PDF" as the destination
  4. Adjust settings if needed (margins, headers, etc.)
  5. Click "Save"

Method 3: Batch Conversion (Advanced)

Use a headless browser or automation tool to batch convert all files:

# Using Chrome/Chromium from command line
for file in html-docs/*.html; do
  chromium --headless --disable-gpu --print-to-pdf="$file.pdf" "$file"
done
